# NREL PVWatts State Averages

US state-average solar peak sun hours reference data from NREL National Solar Radiation Database.

## Files

- `pvwatts-state-averages.json` — state-level peak sun hours + federal ITC

## Source

[NREL PVWatts](https://pvwatts.nrel.gov/) — official US solar production calculator
[NREL NSRDB](https://nsrdb.nrel.gov/) — solar irradiance database
